# MetroMonopoly 捷運大富翁轉盤

一個 **現代簡約風格** 的轉盤抽獎小網站，主題是 **台北捷運 / 新北捷運 / 桃園機場捷運「捷運大富翁」隨機抽站**。  
支援 **RWD（Responsive Web Design）**，可在手機與桌機瀏覽器中直接使用。

> 提醒：本站台僅供遊戲 / 練習用途，站名清單非官方資料，實際營運資訊請以官方公告為主。

---

## 功能特色

- **捷運大富翁轉盤**
  - 從台北捷運、新北捷運，以及桃園捷運機場線目前通車路線中，隨機抽出一個車站。
  - 可同時勾選多條路線，從所有勾選的站點中抽一站。

- **路線篩選**
  - 左側提供多條路線的勾選區：
    - 台北捷運：淡水信義線、松山新店線、中和新蘆線、板南線、文湖線…
    - 新北捷運：環狀線、淡海輕軌、安坑輕軌…
    - 桃園捷運：機場捷運…
  - 只會從「有勾選」的路線中抽選站點。

- **現代簡約 UI + 深色主題**
  - 深色玻璃質感卡片式設計。
  - 轉盤配色簡約，使用淡色系捷運線顏色搭配光影效果。
  - 抽獎完成後，會跳出一個結果小視窗顯示中獎站名。

- **RWD 響應式設計**
  - 小螢幕（手機）採上下排版。
  - 大螢幕（桌機 / 筆電）採左右並排排版。

## 使用方式

1. 將整個專案資料夾（`MetroMonopoly/`）放在任意位置。
2. 用瀏覽器開啟 `index.html`：
   - Windows：檔案總管中雙擊 `index.html`。
   - 或使用 VS Code / Cursor 搭配 Live Server 等外掛啟動本地伺服器。

打開後即可：

1. 勾選要參加抽獎的捷運路線。
2. 按下左側「開始抽獎」按鈕，或點擊轉盤中央的 **Go** 圓形按鈕。
3. 等轉盤轉完，左側會顯示結果，同時彈出結果小視窗。

## 轉盤與抽獎邏輯概述

- 站點資料定義在 `script.js` 最上方的 `metroLines` 陣列中：
  - 每條路線包含：
    - `id`：路線代碼（例如 `"R"`、`"BL"`）
    - `name`：路線名稱（例如 `"淡水信義線"`）
    - `provider`：營運單位（例如 `"台北捷運"`、`"新北捷運"`）
    - `color` / `accent`：在轉盤上使用的顏色
    - `stations`：此路線的站名陣列
- 抽獎流程：
  1. 讀取所有「有勾選」路線，彙整成一個站名清單。
  2. 使用 CSS `conic-gradient` 生成彩色扇形轉盤。
  3. 透過 `transform: rotate(...)` 讓轉盤實際旋轉多圈（約 7～10 圈）。
  4. 根據最後的旋轉角度，計算指針指到哪一個扇形，即為抽中的站點。

---

## 如何調整路線與站點

1. 開啟 `script.js`。
2. 找到 `const metroLines = [ ... ]`。
3. 依照既有格式：
   - **新增路線**：新增一個物件。
   - **調整站點**：在某條路線的 `stations` 陣列中增減 / 修改站名。
4. 儲存後重新整理網頁即可看到更新。
